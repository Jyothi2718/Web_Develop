About Me

Bootstrap is a powerful, feature-packed frontend toolkit. Build anything—from prototype to production—in minutes.

Bootstrap is a free, open source front-end development framework for the creation of websites and web apps. Designed to enable responsive development of mobile-first websites, Bootstrap provides a collection of syntax for template designs.

Bootstrap is the most popular HTML, CSS and JavaScript framework for developing a responsive and mobile friendly website. It is absolutely free to download and use. It is a front-end framework used for easier and faster web development.

Bootstrap provides three types of form layouts: Vertical form (this is default) Horizontal form. Inline form.

Advantages of Bootstrap: Easy to use: Anybody with just basic knowledge of HTML and CSS can start using Bootstrap.
Responsive features: Bootstrap's responsive CSS adjusts to phones, tablets, and desktops.
Mobile-first approach: In Bootstrap 3, mobile-first styles are part of the core framework.

Disadvantages of Bootstrap: It requires that you know how to code. If you don't know how to code at all, you're going to be lost with Bootstrap. ... Its naming scheme can be confusing. It may be less streamlined than you desire. You may become too reliant on it.
