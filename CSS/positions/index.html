Positions in CSS

  1. Static : The original or default position of an element.
  2. Relativen: Element is positioned relative to it's normal/original position. Same as static positioning, just that here we can move elements by adding offset values. Elements stays in normal document flow.
  3. Absolute : Elements is positioned relative to its's parent. If parent is no present, then element is positioned relative to the main document. Element is removed from the normal document flow, no space is created for that element.
  4. Fixed : Element is positioned relative to the viewport. Element doesn't move and stays in the same position even after scrollling. Element is removed from the normal flow.
  5. Sticky : Element is positioned according to the normal flow. Till hits the top, it has normal position, after it touches the top it becomes fixed.
